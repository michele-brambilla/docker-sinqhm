SINQHM on Linux-RTAI Index Page

This is the WWW-server of a histogram memory computer. If you do not know what all this is about or you are not authorized or qualified to handle this: KEEP AWAY! If you are still here, you can try the following things:

  • Show the Histogram Memory Configuration
  • Generate test data. The histogram memory bins will be enumerated.
  • Download histogram memory data
  • Show Status and Counters in HTML
  • Show Status and Counters in Plain Text
  • Show current DAQ Status in Plain Text
  • Start Data Aquisition
  • Stop Data Aquisition
  • Show Debug Buffer
  • AppWeb Error Log
  • AppWeb Access Log
  • Browse Filesystem

Histogram memory configuration happens through XML-files. Suitable XML files can be uploaded to this server through the command: 

wget -S --post-file=your-wonderful-new-configuration.xml http://this.host/admin/configure.egi
wget stores the servers answer to your configuration file into a file called configure.egi. Look there for success or error messages. wget should be available on any properly installed unix system. In the sad case that you have to use windows: try cygwin.
